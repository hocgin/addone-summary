# Popup 模型初始化检查 - 完成总结

## ✅ 功能实现完成

Popup 现在会自动检查模型初始化状态，在未完成时引导用户完成设置。

## 🎯 功能概览

### 三种 Popup 状态

| 状态 | 触发条件 | 显示内容 |
|------|---------|---------|
| **检查中** | Popup 刚打开 | 加载动画 + "正在检查模型状态..." |
| **需要设置** | 模型未初始化 | SetupPromptView（设置提示界面） |

## 🔧 技术实现

### 1. 状态检查

```typescript
// App.tsx
const [isCheckingModel, setIsCheckingModel] = useState(true)
const [needsSetup, setNeedsSetup] = useState(false)

useEffect(() => {
  chrome.runtime.sendMessage({ type: 'CHECK_MODEL_STATUS' }, (response) => {
    setIsCheckingModel(false)

    if (chrome.runtime.lastError || !response?.isModelLoaded) {
      setNeedsSetup(true)
    }
  })
}, [])
```

### 2. 条件渲染

```typescript
// 检查中
if (isCheckingModel) {
  return <div className="app-container checking">...</div>
}

// 需要设置
if (needsSetup && status === 'idle') {
  return <SetupPromptView onSetup={openOnboarding} />
}

// 正常界面
return <WelcomeView ... />
```

### 3. 打开引导页面

```typescript
const openOnboarding = () => {
  chrome.tabs.create({
    url: chrome.runtime.getURL('src/onboarding.html')
  })
}
```
