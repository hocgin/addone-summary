# Chrome 扩展加载测试指南

## 步骤 1: 确认构建完成

```bash
cd /Users/hocgin/Projects/web-llm-x
pnpm build
```

确认 `dist/` 目录包含以下文件：
```
dist/
├── manifest.json          # 扩展配置
├── background.js          # Service Worker (约 6MB)
├── popup.js              # Popup UI (约 150KB)
├── popup.css
├── popup.html
├── icons/                # 扩展图标
├── src/
│   └── popup.html
└── ...
```

## 步骤 2: 在 Chrome 中加载扩展

### 方法 A: 图形界面加载（推荐）

1. **打开扩展管理页面**
   - 在 Chrome 地址栏输入：`chrome://extensions/`
   - 或通过菜单：更多工具 → 扩展程序

2. **启用开发者模式**
   - 点击右上角的"开发者模式"切换开关

3. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 在文件选择器中导航到：`/Users/hocgin/Projects/web-llm-x/dist`
   - 点击"选择文件夹"

4. **确认加载成功**
   - 扩展列表中应出现"网页内容总结 - WebLLM-X"
   - 扩展卡片上显示版本号 "0.1.0"
   - Chrome 工具栏中出现扩展图标（紫色 AI 图标）

### 方法 B: 命令行快速加载（macOS）

```bash
# 打开 Chrome 并加载扩展
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --load-extension=/Users/hocgin/Projects/web-llm-x/dist
```

## 步骤 5: 调试

### 查看 Service Worker 日志

1. 访问 `chrome://extensions/`
2. 找到"网页内容总结 - WebLLM-X"
3. 点击 "Service Worker" 链接
4. 查看控制台日志

**预期日志：**
```
[WebLLM-X Extension installed]
WebLLM-X Content Script loaded
[WebLLM] 开始初始化模型...
[WebLLM] 进度: 5.0% | 阶段: 初始化 WebGPU 环境
[WebLLM] 进度: 45.0% | 阶段: 下载模型权重文件
[WebLLM] 模型初始化完成
```

### 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 扩展图标不显示 | 未正确加载 | 重新加载 dist 目录 |
| 点击图标无反应 | Popup HTML 路径错误 | 检查 manifest.json |
| "WebGPU 不可用" | Chrome 版本过低 | 更新 Chrome 到 113+ |
| 模型下载失败 | 网络问题 | 检查网络连接 |
| 一直显示加载中 | WebLLM 初始化卡住 | 检查 Service Worker 日志 |
| 无法提取页面内容 | Content Script 注入失败 | 检查页面权限 |

### 错误日志示例

**正常日志：**
```
✓ WebLLM-X Extension installed
✓ WebLLM-X Content Script loaded
✓ [WebLLM] 开始初始化模型...
✓ [WebLLM] 模型初始化完成
```

**错误日志：**
```
✗ WebGPU not available
✗ Failed to inject content script
✗ Model load failed
✗ Summarization error: ...
```

## 步骤 6: 重新加载更新

当你修改代码后：

1. **重新构建**
   ```bash
   pnpm build
   ```

2. **在 Chrome 中重新加载**
   - 访问 `chrome://extensions/`
   - 找到扩展
   - 点击刷新图标 🔄

**快捷方式：**
- 在 `chrome://extensions/` 页面按 `Cmd+R` (macOS) 或 `Ctrl+R` (Windows)

## 性能基准

**首次使用（需下载模型）：**
- 网络下载: 2-10 分钟（取决于网络速度）
- 总耗时: 3-15 分钟

**后续使用（模型已缓存）：**
- 模型加载: 5-10 秒
- 内容提取: <1 秒
- 摘要生成: 5-15 秒
- 总耗时: 10-30 秒
