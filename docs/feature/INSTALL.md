# 安装和使用指南

## 安装到 Chrome

### 方法 1：开发者模式加载（推荐用于开发）

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目的 `dist` 目录

### 方法 2：打包安装

1. 首先构建项目：`pnpm build`
2. 在 `chrome://extensions/` 中点击"打包扩展程序"
3. 选择项目的 `dist` 目录
4. 将生成的 .crx 文件拖入 Chrome

## 使用方法

1. 浏览任意网页
2. 点击浏览器工具栏中的扩展图标
3. 点击"开始分析"按钮
4. 等待 AI 分析完成（首次使用需要下载模型）

## 首次使用

首次点击时，扩展会自动下载 AI 模型（约 600MB-1GB）。下载时间取决于您的网络速度。

## 系统要求

- Chrome 113 或更高版本
- 支持 WebGPU 的硬件
- 至少 2GB 可用内存

## 故障排除

### 扩展图标显示但点击无反应

1. 检查控制台错误（右键点击扩展图标 -> 检查弹出窗口）
2. 确保已在 `chrome://extensions/` 中启用扩展

### "WebGPU 不可用"错误

1. 更新 Chrome 到最新版本
2. 在 Chrome 地址栏输入 `chrome://flags/`
3. 搜索 "WebGPU" 并确保已启用
4. 重启浏览器

### 模型下载失败

1. 检查网络连接
2. 确保有足够的磁盘空间
3. 尝试关闭并重新打开扩展

### 内存不足错误

1. 关闭其他浏览器标签页
2. 关闭其他占用内存的应用程序
3. 尝试在较短的文章上测试
